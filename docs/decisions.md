| 2026-07-05 | Naming pass complete: final name is **Borrow Native** (slug `borrow-native`) | Chosen by coderturtle from naming-agent candidates (Borrow Native, Fearless Loop, Rust, Verified, Compile or Concede) — all four confirmed available under `github.com/coderturtle` before presenting. | 2026-07-05 | Second real run of the Workshop Gremlin (`~/hekton/gremlins/workshop/workshop-gremlin.md`), first on a subject other than agentic engineering | Deliberately chosen as this Gremlin's next run per its own Follow-Up Actions ("watched across 2-3 more workshops before either claims v1 stability") and `terminal-velocity`'s note that this Gremlin had only run once, entirely on its own founding subject. Surfaced a genuine augmentation before any content was built: Rust has an objective checker (compiler/`cargo test`/`cargo clippy`) `terminal-velocity`'s subject never had, making a **deterministic gate tier** a strict upgrade over Coachgremlin's purely subjective rubric — written back into `workshop-gremlin.md` (Design Principle 2 extension + new "Variant: Tech/Language Workshops" section), `coachgremlin.md` (two-tier grading in Workflow step 3), and `workshop-review-panel.md` (Instructional Designer persona check), each as a minor version step, not a new roster item. |
| 2026-07-05 | 6-module + capstone arc, anchored to the Rust Book/Rustlings/Exercism (all fetched directly, not recalled from memory) | Rustlings' exercise-to-chapter mapping and Exercism's `config.json` prerequisite arrays gave real, checkable evidence for a concept-dependency graph rather than an invented sequence. Iterators/Closures folded into Module 04 as an optional extension rather than a standalone module, reusing `terminal-velocity`'s validated "required core + optional extensions" shape. |
| 2026-07-05 | Ran the Workshop Review Panel (7 personas) against the naming + design docs before building the module skeleton | Matches `terminal-velocity`'s validated timing (cheapest to fix while still prose).
